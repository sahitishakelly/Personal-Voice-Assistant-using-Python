# Personal Voice Assistant using Python

A Python-based Personal Voice Assistant that uses Speech Recognition and Text-to-Speech technologies to perform basic desktop and web tasks through voice commands.

## Features

* Speech-to-Text using SpeechRecognition
* Text-to-Speech using pyttsx3
* Open desktop applications
* Search the web using voice commands
* Tell current time
* Wikipedia search support
* Rule-based command mapping
* Robust error handling

## Technologies Used

* Python
* SpeechRecognition
* pyttsx3
* PyAudio
* Wikipedia API

## Sample Commands

* Open Notepad
* Open Calculator
* Open Paint
* Search Python Programming
* What is the time
* Wikipedia Artificial Intelligence
* Exit

## Installation

Install required libraries:

pip install SpeechRecognition pyttsx3 pyaudio wikipedia

## How to Run

1. Save the Python file
2. Run the file using IDLE or terminal
3. Speak commands through microphone

## Project Objective

This project demonstrates how voice-based assistants work using Python by integrating speech recognition, command processing, and voice response systems.
